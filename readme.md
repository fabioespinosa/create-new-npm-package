# create-new-npm-package

> Handles creating readme.ms, LICENSE, .gitignore, package.json, tes.js and index.js for you.

So you can actually spend your time coding your package.

## Install

Install globally:
\`\`\`
\$ npm install create-new-npm-package -g
\`\`\`

## Usage

\`\`\`
\$ create-npm-package
\`\`\`
