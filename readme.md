# create-new-npm-package

> Handles creating readme.md, LICENSE, .gitignore, package.json, test.js and index.js for you.

So you can actually spend your time coding your package.

## Install

Install globally:

```
$ npm install create-new-npm-package -g
```

## Usage

```
$ create-npm-package
```
